# Atlas Trips

http://atlas-trips.herokuapp.com/

Collaborative trip planning tool

A combination web and mobile app that can help streamline planning a trip with a group. Users can invite friends to a trip, and edit details such as activities, accommodations, and travel information. Built using NodeJS, Express, Postgres, Sequelize, React, Redux, React-Native, Nodemailer, Google Maps API, & Socket.IO
